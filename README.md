# Casualties-Unknown-Text-Tweaks

这是一个针对游戏《未知伤亡》（Casualties Unknown）开发的界面显示与液体效果辅助计算插件。

本插件旨在解决游戏中液体成分过多时悬浮窗（Tooltip）排版溢出、超出边界的问题，并通过读取外部 XML 配置文件，自动换算并展示不同摄入途径下的预估数值影响及安全剂量警示。

---

## 主要功能

* **多成分分页浏览**
  当容器内含有多种液体成分时，在悬浮窗显示状态下按住 `Shift` 键并滚动鼠标滚轮，即可分段切换浏览各个成分的详细说明，避免文本内容超出屏幕。

* **多途径效果实时预估**
  插件会自动根据配置计算液体在“口服”、“静脉注射”、“外敷”三种不同使用途径下的预期影响，直观展示水分、饱腹度、疼痛值、免疫时间等参数的变化。

* **持续性效果（Buff）与副作用预览**
  自动换算增益或副作用的持续时间，并换算展示每秒每毫升的具体作用率。

* **摄入剂量与过量风险警示**
  结合容器特性（如喷雾、血袋、药瓶等单次最大摄入量），自动折算预估吸入量。一旦预估摄入量超出配置的安全阈值，悬浮窗将自动生成红色的风险预警文字。

* **纯数据驱动的自定义配置**
  插件所有的计算规则、数值折算比例以及警示文本均由外部 XML 配置文件驱动。玩家可以随时修改、添加自定义的液体及其对应的数学转换规则（支持线性与反比规则）。

## 安装指南（新手向）

本模组已打包为标准目录结构的整包。安装前请确保您的游戏已经安装了 **BepInEx** 运行框架（通常要求 5.x 版本）。

### 第一步：定位游戏根目录（Steam 玩家）

1. 打开 Steam 客户端并进入“库”。
2. 在左侧游戏列表中找到并右键点击《未知伤亡 试玩版》（Casualties Unknown Demo）。
3. 在弹出的菜单中，依次选择 **“管理”** -> **“浏览本地文件”**。
4. 电脑会自动打开一个文件夹，该文件夹即为游戏的**根目录**（内含游戏主程序 `CasualtiesUnknown.exe`）。

### 第二步：安装模组

1. 下载本模组发布的最新整合压缩包。
2. 将压缩包内的所有内容解压。
3. 把解压出来的 `BepInEx` 和 `CasualtiesUnknown_Data` 两个文件夹直接拖入刚刚打开的游戏根目录中，提示重复时选择覆盖合并。
4. 启动游戏即可生效。

### 安装结果确认
若覆盖正确，相应文件的实际路径应如下：
* **插件文件**：`[游戏根目录]/BepInEx/plugins/文本增强.dll`
* **配置文件**：`[游戏根目录]/CasualtiesUnknown_Data/Lang/文本增强-配置.xml`

## 游戏内使用说明

* **默认预览模式**
  鼠标悬停在液体或容器上方时，会显示合并后的液体成分。如果检测到配置数据，下方会显示在当前容器限制下，一次性摄入该液体所预估产生的影响。

* **详细分页模式**
  1. 将鼠标悬停在目标物品上。
  2. **按住键盘的 `Shift` 键不放**。
  3. 此时悬浮窗会显示当前页面的液体成分。**滚动鼠标滚轮**可以向后或向前翻页，查看其他液体成分的具体明细。

## 高级自定义指南（XML 配置说明）

如果您需要自行微调数值平衡或添加新的 MOD 液体，可以直接用文本编辑器（如 Notepad++）打开 `文本增强-配置.xml` 进行修改。

### 核心节点说明

```xml
<Liquid Name="液体名称">
  <!-- 口服途径对应的属性变化 -->
  <Oral>
    <Stats>
      <!-- Field: 属性名; ValuePerML: 每毫升增加的值; ValueColor: 文本颜色 -->
      <Stat Field="水分" ValuePerML="0.09" ValueColor="#00FF00" />
      <Stat Field="体温" ValuePerML="-0.0025" ValueColor="#00FF00" Suffix="°C" />
    </Stats>
  </Oral>
  <!-- 剂量预警 -->
  <DosePrompts>
    <DosePrompt Threshold="80.0" Message="警示文本内容" />
  </DosePrompts>
</Liquid>
```

### 进阶：数学公式规则（Rule）
对于非线性变化的属性（例如部分液体在特定毫升数内有效果上限，或者按非线性比例计算），可以使用 `<Rule>` 节点：

```xml
<Stat Field="局部消毒" ValueColor="#00FF00" Suffix="秒">
  <!-- Mode="Linear"（线性比例）: 结果 = 毫升数 * Multiplier -->
  <!-- UseMaxLimit="true"（限制最大值）: 最终计算结果不超过 MaxLimit (此例为350) -->
  <Rule Mode="Linear" Multiplier="3.5" UseMaxLimit="true" MaxLimit="350" />
</Stat>
```

* **Linear 模式**：
  $$\text{计算值} = \text{基础值} + (\text{乘数} \times \text{实际毫升数})$$
* **Inverse 模式**：
  $$\text{计算值} = \text{基础值} + (\text{乘数} / \text{实际毫升数})$$

## 常见问题排查

* **为什么游戏里没有显示预估数值？**
  * 请确认 XML 配置文件其所在的路径是否准确（必须位于游戏 Data 目录下的 `Lang` 文件夹内）。
  * 检查 `BepInEx/LogOutput.log`，如果日志中出现 `[TooltipFix] Config Error`，说明 XML 文件存在标签未闭合等语法错误。

* **为什么滚动鼠标滚轮无法翻页？**
  * 翻页功能必须在**按住键盘 Shift 键**的同时滚动轮滚才会触发。
  * 确认该容器中是否确实包含至少两种及以上的液体成分。
